This program finds the fastest DNS servers for your internet connection. Slow DNS settings lead to delay when you open websites. This tool tests servers locally on your computer to save you time. It helps users of V2Ray and Sing-box identify reliable connections.

*   An active internet connection.
*   No special software installation is required.

## 🛠 How to Use the Scanner

When the window opens, you see a list of available servers. The application performs a scan automatically. Please wait for the green bars to appear next to each server name.

*   **Server List:** This shows the address of the DNS servers tested.
*   **Latency:** This column shows how fast each server responds. Low numbers are better for gaming and browsing.
*   **Status:** A checkmark means the server is ready for use.

To update the list, press the Refresh button. You can right-click any result to copy the server address for use in your V2Ray or Sing-box configuration file.

## 📊 Understanding Your Results

Results show your current connection quality. A lower millisecond (ms) value indicates a faster response time. If you experience slow page loads, look for servers with the lowest latency in the list.

## 🖥 Technical Details

The scanner uses common network protocols to measure round-trip time. It pings various DNS providers to calculate the connection delay. This method mimics how your browser finds websites. By comparing different providers, you ensure the best performance for your proxy setup.
